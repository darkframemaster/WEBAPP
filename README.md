# WEBAPP
A web app using python. 

###about the dirs.
```
WEBAPP/                  <-- 根目录
|
+- backup/               <-- 备份目录
|
+- conf/                 <-- 配置文件
|
+- dist/                 <-- 打包目录
|
+- www/                  <-- Web目录，存放.py文件
|  |
|  +- static/            <-- 存放静态文件
|  |
|  +- templates/         <-- 存放模板文件
|
+- LICENSE               <-- 代码LICENSE
```

### how to use.
you shoud do some changes in `config.py` and `schema.sql` in `www/`.

run the script `app.py` to start server ,see `readme.mkd` in `www/`.

if you using windows you may need do some changes in code.
